# nginx + PHP FPM Docker container

Based on the official [PHP Alpine container](https://hub.docker.com/_/php) to run Magento2 on top of it.
It uses Supervisord (http://supervisord.org/) to start nginx, PHP FPM, Postfix and a cron daemon.

Container images can be downloaded from [DockerHub](https://hub.docker.com/r/phoenixmedia/nginx-php/tags).
